# SOffline Intelligent Voice-Controlled Home Control System Based on STM32 & ASRPRO

# STM32-ASRPRO 智能语音管家家居控制系统
**Offline Intelligent Voice-Controlled Home Control System Based on STM32 & ASRPRO**

---

## 📌 项目简介
本项目是一套**模块化、低成本、离线可用**的嵌入式智能家居控制系统，以 **STM32F103C8T6** 为主控核心，搭配 **ASRPRO离线语音识别模块**，实现语音指令控制、环境监测、异常报警、OLED状态显示等功能。
系统无需联网、响应快、隐私性强，可直接控制灯光、风扇等家居设备，同时监测温湿度、烟雾、酒精浓度与人体感应，适合家庭、宿舍等轻量化智能改造场景。

## ✨ 核心功能
- 🎤 **离线语音识别**：本地识别“打开灯光”“关闭风扇”等自定义指令，识别准确率 ≥97.3%
- 🌡️ **多维度环境监测**：DHT11温湿度 + MQ-2烟雾 + MQ-3酒精 + SR602人体感应
- ⚡ **设备本地控制**：通过继电器模块控制灯光、风扇等220V家用设备
- 🔔 **异常声光报警**：浓度超标自动报警，蜂鸣器+指示灯+语音同步提醒
- 📺 **OLED实时显示**：0.96寸屏展示环境数据、设备状态、指令反馈
- 🧩 **模块化扩展**：支持自由添加传感器/执行器，无需重构系统

## 🛠️ 硬件平台
| 模块类型 | 型号/组件 | 功能说明 |
|--------|-----------|----------|
| 主控芯片 | STM32F103C8T6 | ARM Cortex-M3，72MHz，处理核心 |
| 语音识别 | ASRPRO | 离线语音识别，低延迟、高准确率 |
| 温湿度传感器 | DHT11 | 室内温度、湿度采集 |
| 气体传感器 | MQ-2、MQ-3 | 烟雾、酒精浓度检测与报警 |
| 人体感应 | HC-SR602 | 人体红外检测，智能安防联动 |
| 执行器件 | HW-307继电器 | 灯光、风扇等设备开关控制 |
| 显示模块 | 0.96寸 OLED（SSD1306） | I2C通信，状态与数据可视化 |

## 🧠 软件架构
系统采用**分层模块化设计**，代码结构清晰、易维护、易扩展：
1. **感知层**：传感器数据采集、滤波、校准
2. **控制层**：语音指令解析、异常逻辑判断、设备调度
3. **执行层**：继电器控制、报警输出、OLED显示刷新
4. **交互层**：语音反馈、界面显示、状态同步

核心算法：
- 滑动平均滤波：平滑传感器数据，降低噪声干扰
- 动态时间规整(DTW)：提升语音指令匹配精度
- 中断+轮询双模式：保证系统实时性与稳定性

## 📊 系统测试与性能
在家庭环境（3m距离、75dB噪声）下实测：
- 语音识别准确率：**97.3%**
- 指令响应时间：**≤ 0.5s**
- 温湿度误差：**±1℃ / ±3%RH**
- 报警响应时间：**≤ 1s**
- 待机功耗：**≤ 50mA**
- 工作功耗：**≤ 200mA**
- 稳定性：**72小时连续运行无死机、无误动作**

## 🚀 快速使用
1. **硬件接线**：按照 `Hardware/` 目录接线图完成模块连接
2. **环境搭建**：安装 Keil MDK、STM32CubeMX、CH340 驱动
3. **代码编译**：打开工程文件，编译无误后下载至 STM32
4. **上电运行**：系统自动初始化，语音指令即可控制设备
5. **状态查看**：OLED 实时显示环境数据与设备状态
